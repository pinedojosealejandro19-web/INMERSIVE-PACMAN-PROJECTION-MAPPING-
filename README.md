# INMERSIVE-PACMAN-PROJECTION-MAPPING-
Inmersive Pacman is a projection-mapping version of the classic game built with HTML5, CSS, and JavaScript. It runs in large immersive spaces and uses a QR-based mobile controller for real-time input. The project is open source and currently in active development.

Inmersive Pacman is a projection-mapping adaptation of the classic arcade game, designed for large-scale immersive environments instead of traditional screens. The project transforms Pac-Man into a spatial experience where gameplay extends across multiple surfaces, using a custom tile-based engine optimized for non-standard aspect ratios and multi-panel layouts.

The system integrates a web-based game built in HTML5 Canvas with a real-time mobile controller connected through Firebase. Players interact using a phone as a wireless gamepad via QR connection, sending directional and action inputs to the projected game in real time. This architecture allows flexible deployment in events, installations, and interactive exhibitions.

The purpose of this project is to recreate and make accessible immersive experiences similar to large-scale installations like the Telfair Museum 3D Pac-Man. It is open source, and any modifications or improvements are intended to remain public and free.

The game includes smooth grid-based movement, portal mechanics, sprite animation, and scalable rendering adapted for projection setups, bridging classic gameplay with modern interactive technology.
